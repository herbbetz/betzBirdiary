The [Birdiary project](https://github.com/Birdiary/station) at Münster University uses a video camera in a [birdhouse](docs/birdhouse/birdhouse.md) to identify garden birds. The bird videos are collected on the [Birdiary platform](wiediversistmeingarten.org/view/station/87bab185-7630-461c-85e6-c04cf5bab180).

In early 2024, I started developing my own software version *betzBirdiary* based on the *station120423.img* file image from the university for the Raspberry 4B and the "Raspicam v3". My goal was to create a minimalist, modular version that would make it easy for me to experiment and update to new versions such as Raspberry Trixie. I also wanted a web interface to view my station in a browser and on my smartphone.

You can find the Release of my image in the [cloud](https://drive.google.com/drive/folders/11WduKyMzzzmW61bC7l0BlDjjx6e_ImHC) and install it in the same way as the usual image of *Raspberry Trixie*.
